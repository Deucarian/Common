# Support for Deucarian Common

Use [GitHub Issues](https://github.com/Deucarian/Common/issues) for reproducible defects in the small shared runtime primitives, documentation gaps, and narrowly scoped feature requests. Search existing issues before opening a new one.

Include the package version, Unity editor version, affected primitive, the smallest reproduction, expected and actual results, and the full error or stack trace.

`main` is the stable package channel. `develop` is a preview channel and may change before promotion. Please reproduce against the applicable current channel before reporting an older snapshot.

Do not disclose vulnerabilities or private project data in an issue. Follow [SECURITY.md](SECURITY.md) for private security reports.

Support is provided on a best-effort basis. Requests that turn Common into a broad utility collection or duplicate another package's capability are out of scope.
